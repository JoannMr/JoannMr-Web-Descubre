¿Cómo funciona esta web?

Esta web te permite explorar información detallada sobre cualquier población de España. Para comenzar, sigue estos pasos:

  1. Selecciona una Comunidad Autónoma: Elige una comunidad de la lista desplegable.
  2. Selecciona una Provincia: Una vez seleccionada la comunidad, elige una provincia.
  3. Selecciona una Población: Finalmente, selecciona la población que deseas explorar.
  4. Haz clic en "BUSCAR": La web te mostrará información sobre la población, incluyendo datos de Wikipedia, el clima actual, imágenes relacionadas y un mapa interactivo.
  5. Búsqueda por voz: También puedes utilizar el botón de búsqueda por voz para seleccionar la comunidad, provincia y población hablando.

¡Es así de sencillo! Explora y descubre todo lo que necesitas saber sobre cualquier población de España.

APIs Utilizadas

  • Wikipedia API:

    Utilitzem l'API de Wikipedia per obtenir informació detallada sobre la població seleccionada. Quan l'usuari tria una població, fem una consulta a l'API per obtenir un resum introductori (extracte) de la pàgina de Wikipedia corresponent.

  • Open-Meteo API:

    Aquesta API ens proporciona dades meteorològiques en temps real. Utilitzem les coordenades de la població seleccionada per obtenir la temperatura actual i la velocitat del vent.

  • Wikimedia Commons API:

    Amb aquesta API, obtenim imatges relacionades amb la població seleccionada. Les imatges es mostren en una galeria perquè l'usuari pugui veure aspectes visuals del lloc.

  • Nominatim (OpenStreetMap):

    Utilitzem Nominatim per obtenir les coordenades geogràfiques (latitud i longitud) de la població seleccionada. Aquestes coordenades es fan servir per mostrar un mapa interactiu amb la ubicació exacta.

  • Web Speech API:

    Aquesta API permet als usuaris realitzar cerques per veu. Quan l'usuari prem el botó de "Buscar per veu", pot dir el nom de la comunitat autònoma, la província i la població que vol explorar. El navegador reconeix la veu i completa automàticament els camps del formulari.
