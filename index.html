¿Cómo funciona esta web?

Esta web te permite explorar información detallada sobre cualquier población de España. Para comenzar, sigue estos pasos:

  1. Selecciona una Comunidad Autónoma: Elige una comunidad de la lista desplegable.
  2. Selecciona una Provincia: Una vez seleccionada la comunidad, elige una provincia.
  3. Selecciona una Población: Finalmente, selecciona la población que deseas explorar.
  4. Haz clic en "BUSCAR": La web te mostrará información sobre la población, incluyendo datos de Wikipedia, el clima actual, imágenes relacionadas y un mapa interactivo.
  5. Búsqueda por voz: También puedes utilizar el botón de búsqueda por voz para seleccionar la comunidad, provincia y población hablando.

¡Es así de sencillo! Explora y descubre todo lo que necesitas saber sobre cualquier población de España.

APIs Utilizadas

  • Wikipedia API:

    Utilizamos la API de Wikipedia para obtener información detallada sobre la población seleccionada.

  • Open-Meteo API:

    Proporciona datos meteorológicos en tiempo real usando las coordenadas de la población seleccionada.

  • Wikimedia Commons API:

    Obtenemos imágenes relacionadas con la población seleccionada para mostrarlas en la galería.

  • Nominatim (OpenStreetMap):

    Permite obtener las coordenadas geográficas de la población y mostrar el mapa interactivo.

  • Web Speech API:

    Facilita la búsqueda por voz para seleccionar la comunidad, provincia y población mediante comandos de voz.
